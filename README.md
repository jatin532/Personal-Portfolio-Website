# Jatin Chaudhary Portfolio

This is a personal portfolio website built with HTML, CSS, and JavaScript. It showcases the developer's introduction, skills, projects, education, and contact information in a modern and responsive layout.

## Features

- Responsive navigation bar
- Hero section with animated typing effect
- About section with profile information
- Skills section with icons
- Projects and education sections
- Contact section
- Smooth scrolling and subtle animations

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Font Awesome icons
- Google Fonts

## Project Structure

- index.html - Main page structure
- style.css - Styling and responsive design
- script.js - Interactive effects and animations
- images/ - Project images and assets

## How to Run

1. Open the project folder in your browser.
2. Double-click index.html, or use a live server extension such as Live Server in VS Code.

## Deployment on GitHub Pages

1. Create a new GitHub repository.
2. Upload these files to the repository.
3. Go to Settings > Pages.
4. Select the main branch as the source.
5. Your portfolio will be available at:
   https://your-username.github.io/your-repository-name/
